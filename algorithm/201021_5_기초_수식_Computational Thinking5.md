### 연습문제: 다음 재귀식들을 O() notation 수준으로 풀어라. 

재귀식은 점화식이 아니다.

무슨 뜻이냐면, T(n) = n개 만큼 들어왔을 때 연산량

T(n) = T(n-1) + 1의 뜻 : n개 만큼 들어왔을 때의 연산량이, n-1개만큼 들어왔을 때의 연산량보다 하나 많다. 

#### 문제1

![image-20201022133859372](images/image039.png) 

문제 

(n=0일 때) T(0) = 1

(n != 0일 때) T(n) = T(n-1) + 1



 풀이

T(n) = T(n-1) + 1

T(n-2) + 1 +1 

= T(n-k) + k

=T(0) + n

= 1 + n

-> O(n)
